# CS Zero to Hero

A structured teaching repository covering computer science fundamentals — data structures, algorithms, and core theory. Part of the Zero to Hero course family.

> **Status:** This repository is in development. Content is being built out from existing course material.

---

## Who Is This For?

- Developers who want to strengthen their CS foundations
- Delegates preparing for technical interviews or career progression
- Learners who have completed a language-focused course (Python or Java Zero to Hero) and want to go deeper on theory

---

## Planned Topics

- Data structures — arrays, linked lists, stacks, queues, trees, graphs, hash maps
- Algorithms — searching, sorting, recursion, dynamic programming
- Complexity analysis — Big O notation, time and space tradeoffs
- Problem-solving patterns — common approaches and when to apply them
